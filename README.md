## 👵🏻MVP
1️⃣ 사용자가 위치한 지역의 구인구직 정보를 제공합니다.  
2️⃣ 사용자가 원하는 구직 분야를 필터링할 수 있습니다.  
3️⃣ 직관적으로 확인할 수 있는 구직 서비스를 제공합니다.  

<br/>

## 👴🏻Tech Stack
**Web Frontend**  
`React.js`, `JavaScript`, `Kakao Local API`  
  
**Server**  
`Node.js`, `Express`, `JavaScript`, `PostgreSQL`, `DataGrip`, `Naver Cloud SMS API`
